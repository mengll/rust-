### rust 中的对象，解结构赋值到，对应的参数接收位置
```
struct P(f32,f32,f32);
fn calc(arg:P)->f32 {
    let P(x,_,y) = arg;
    x*x +y*y
}

fn func(P(x,_,y):P) ->f32{
    x*x + y*y
}
// 对象的析构的方法
fn main(){
    let t = P(1.0,2.0,3.0);
   // println!("{}",calc(t));
    println!("{}",func(t)); // 一个对象，只能被一个对象，或函数使用
}

可以使用 .. 标识其它的参数  _ 占位符，无法赋值，取值
fn main(){
 let x =(12,23,34);
 let (a,..,_y) = x;
  println!("{}--{}",a,_y);
}

```
