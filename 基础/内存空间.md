### rust 中对象内存的分配
```
struct A{
  a:u32,
  b:Box<u64>,
}
enum E {
  H(u32), //E: 4
 // M(Box<u32>) // 16
}

fn main(){
  println!("A: {:?}", std::mem::size_of::<A>()); //16
  println!("E: {:?}", std::mem::size_of::<E>()); // 4
}

// 解析当前A 中最长的成员b最长，占8个字节，所以按照8个自己对齐 ，当结构A在函数中有实例被初始化是，该结构体会被放到栈中，首地址为第一个成员变量a
// 的地址 Box<u32> 类型会在堆上开辟空间存放数据，指针会返回给成员b 并存放在栈中 一共占了8个字节  
```
